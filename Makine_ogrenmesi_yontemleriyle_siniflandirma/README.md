# Makine Öğrenmesi Yöntemleriyle Sınıflandırma

Bu proje, meme kanseri veri seti üzerinde Lojistik Regresyon, Random Forest ve SVM modellerinin karşılaştırmalı performans analizini içermektedir.

## İçerik
- Veri önişleme (temizleme, encoding, scaling)
- Eğitim/Test ayırma
- Logistic Regression, Random Forest ve SVM modelleri
- Accuracy, Precision, Recall, F1 Score metrikleri
- Sonuçların `results/` klasörüne kaydedilmesi
- Model dosyalarının `.pkl` formatında kaydedilmesi
